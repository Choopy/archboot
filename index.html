Arch Linux Auto Install

View project on GitHub

This script perform an automatic install of Arch Linux. Pretty simple, it only ask you for hostname, username, password, which disk to use, required partitions size, language & if you wish to install a desktop environment.

Requirements

The only thing you need is one USB bootable device of Arch Linux and one HDD dedicated for the installation (see USB Boot instructions for Windows or for Linux).

Usage

1. Reboot your computer, open Boot Selection Menu and boot on the USB bootable device.

2. On the install menu of ArchLinux, select Boot Arch Linux (x86_64).

3. For AZERTY keyboard run: loqdkeys fr

4. Download archboot and follow the instructions:

wget tiny.cc/archboot; sh archboot

Notes

For list of all available languages and regions codes, please see languages and regions.

If you want to see whose packages are installed by this script, please see Applications Folder.

Available desktop environment: KDE - GNOME - Deepin - Xfce - LXDE

Ability to enable AUR and/or Multilib support is also available.

Logs

Take a look in /var/log/archboot for logs of the installation.

cat /var/log/archboot/archboot_*.log
